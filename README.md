# Hari Sakthivel — The Engineer's Odyssey

A cinematic, PS5-trailer-style portfolio with auto-animated sections.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve hari-portfolio
```

## Assets

Place portrait images at:
- `assets/images/hero-portrait.jpeg`
- `assets/images/engineer-portrait.jpeg`

Fallback monograms display automatically if images are missing.

## Features

- **Hero & Engineer**: Glitch text, hologram frames, parallax, animated stat counters
- **Arsenal**: Auto-deploying chests on scroll (no click required)
- **Work**: Cinematic scroll-driven project cards with letterbox transitions
- **Engineering Lab**: Holographic hex auto-materialization
- **Journey**: Futuristic 3010 timeline with neon energy beam
- **Continuous Learning**: Orbital satellite auto-cycle animation
- **Beyond Engineering**: Includes Assistant Head of Design Team, TED Club
